# FallSafe Dashboard

Professional responsive website for an ESP32 + GSM fall detection device.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy online with Vercel
1. Create a GitHub repository and upload this folder.
2. Go to Vercel.com.
3. Import the GitHub repository.
4. Framework: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.

After deployment, the website will be accessible from any device using the Vercel link.
